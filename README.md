## Overview

This project presents a machine learning and deep learning framework for in-bed human posture classification using pressure sensor grid data from PhysioNet.

## Dataset

- PhysioNet Pressure Map Dataset (PMD)
- 13 participants
- 32×64 pressure sensor grid
- 2048 pressure values per frame

## Methods

- Exploratory Data Analysis (EDA)
- Principal Component Analysis (PCA)
- Pressure Variance Profiling
- Logistic Regression
- Support Vector Machine
- KNN
- Artificial Neural Network (ANN)
- Convolutional Neural Network (CNN)

## Results

| Model | Accuracy |
|---------|---------|
| ANN | 94.2% |
| CNN | 97.5% |

## Applications

- Smart Healthcare
- Elderly Monitoring
- Pressure Ulcer Prevention
- Astronaut Posture Tracking
